\# Görüntüden Çiçek Türü Tanıma Sistemi



Bu proje, kullanıcı tarafından yüklenen çiçek görselini analiz ederek çiçeğin türünü tahmin eden yapay zeka tabanlı bir uygulamadır.



\## Kullanılan Teknolojiler



\- Python

\- TensorFlow / Keras

\- EfficientNetB0

\- Transfer Learning

\- Fine Tuning

\- Streamlit

\- NumPy

\- Pillow



\## Tanınan Çiçek Türleri



\- Papatya

\- Karahindiba

\- Gül

\- Ayçiçeği

\- Lale
